# AMC Pilot Sample Pack Copy

These sample cases show how AMC handles different career decision structures, including comparative path decisions and single-path structural sustainability checks.

### `amc_case_jll_comparative.json`
- Display label: Global Relocation vs Local Continuity
- One-line description: A comparative case showing how family stability, cross-market transition burden, and long-term platform value can pull in different directions.

- Display label: Immediate Reemployment vs Extended Search
- One-line description: A comparative case contrasting near-term income stabilization against longer-horizon repositioning value.

### `amc_case_exit_package_vs_stay.json`
- Display label: Exit Package vs Stay Decision
- One-line description: A comparative case on whether to accept a retirement package now or retain the current corporate path under uncertainty.
- Best for showing: A broadly relatable two-path decision with clear structural tension and practical implications.
