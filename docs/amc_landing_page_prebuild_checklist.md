# AMC Landing Page Pre-Build Checklist

## 1) Purpose
This checklist is the final readiness gate before AMC landing page design/development begins, confirming that core page structure, copy decisions, and sample-pack policy are fixed enough to implement without strategic ambiguity.

## 2) Required Inputs Confirmed
- [x] Landing page section order is fixed.
- [x] Final compact landing copy is available.
- [x] Pilot sample pack policy is fixed.
- [x] Public top-3 sample set is fixed.
- [x] Pilot-only single sample is identified.
- [x] Tier structure is fixed (`Essential`, `Executive`).
- [x] FAQ draft is available.
- [x] CTA options are available.

## 3) Content Decisions Confirmed
- [x] AMC positioning is structure-first.
- [x] AMC is not generic coaching.
- [x] AMC is not generic AI advice.
- [x] Tone is premium, structural, calm, and non-salesy.
- [x] Deprecated offers are not referenced.

## 4) Build Readiness Checks
- [x] Hero section spec is ready.
- [x] Sample card spec is ready.
- [x] Tier section spec is ready.
- [x] FAQ spec is ready.
- [x] Closing CTA spec is ready.
- [x] Handoff document is complete enough for design/dev implementation.

## 5) Open Questions / Optional Follow-Up (Non-Blockers)
- [ ] Decide whether to introduce pricing on the first release page or keep CTA-only.
- [ ] Decide whether to add real sample PDF/DOCX previews in a later iteration.
- [ ] Decide whether initial conversion flow should use direct lead capture or contact CTA only.

## 6) Final Recommendation
Ready to begin landing page build.
