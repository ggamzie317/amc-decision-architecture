# AMC Delivery Modes

## Essential
- Essential includes the report only.
- It is the base delivery mode.
- No follow-up chatbot access is included.

## Executive
- Executive includes the same report plus 7-day report-linked follow-up chatbot access.
- Executive does not change the report itself.
- It adds a bounded interpretation and clarification layer after delivery.

## Executive Chatbot Positioning
The Executive chatbot is:
- a report interpreter layer
- not a recommendation engine
- not open-ended coaching
- not generic motivational support
- limited to report-linked clarification and interpretation

## Future Executive Email Packaging Requirement
Executive delivery should eventually include:
- report delivery
- chatbot access information
- usage window guidance
- scope and boundary explanation

## Operating Principle
Executive should be treated as a delivery package, not just a report attachment.
