# AMC Operator Run Sequence (Minimal)

Use this sequence for one real case from generation to delivery.

## 1) Run Generation
- Follow `docs/amc_dry_run_checklist.md` to generate the report and payload for the target intake.

## 2) Verify Outputs
- Confirm final DOCX and payload JSON files exist at the expected paths.
- Confirm no visible placeholder leakage.

## 3) Complete Delivery Readiness Review
- Run `docs/amc_delivery_readiness_checklist.md`.
- Confirm section order, comparative visibility, fallback acceptability, and restrained tone.

## 4) Prepare Operator Handoff Package
- Fill `docs/amc_operator_handoff_template.md`.
- Confirm attached file list and readiness confirmations are complete.

## 5) Send Final Delivery
- Send the finalized report package.
- Record any follow-up/caveat note for post-delivery support.
